# Contributing

Contributions of any kind are welcome!

Feel free to open an issue to report a bug, suggest an improvement, ask a question, or share an idea.
Pull requests are also appreciated, whether they fix bugs, improve documentation, add features, or clean up code.

Please use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) for pull request titles and commit messages when practical.
Release automation uses those messages to decide the next package version.
Signed commits are encouraged so provenance is clear in the published history.

Thanks for helping make this project better.
